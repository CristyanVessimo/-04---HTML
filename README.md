# Evolução dos Animes Shonen

Nome: CRistyan Véssimo Gomes de Oliveira

Este projeto foi desenvolvido como parte de uma atividade prática de HTML.

O objetivo do site é apresentar a evolução dos animes shonen através de três obras importantes:
- Cavaleiros do Zodíaco
- Dragon Ball Z
- Bleach

O site contém múltiplas páginas, imagens, vídeos e links externos, utilizando conceitos básicos de HTML e CSS.
